# 家で運動したくなるアプリ 設計ドキュメント

**作成日**: 2026-03-29

---

## 概要

家での運動継続を助けるWebアプリ。最大の課題である「やる気が起きない」をゲーミフィケーション（経験値・レベルアップ・キャラクター育成・実績バッジ）で解決する。自分用の個人ツールとしてシンプルに作る。

---

## アーキテクチャ

### 技術スタック
- HTML / CSS / Vanilla JS（フレームワークなし）
- データ永続化：localStorage
- デプロイ：静的ファイルをそのまま配置（GitHub Pages等）

### ファイル構成
```
運動アプリ/
├── index.html   # 全画面のHTMLマークアップ
├── style.css    # スタイル（レスポンシブ対応）
└── app.js       # ゲームロジック・画面切り替え・localStorage
```

### 画面構成（SPA風に切り替え）
1. **ホーム** — キャラクター表示 + 経験値バー + 「運動する」ボタン
2. **運動選択** — プリセットメニュー一覧 → 完了ボタンで経験値獲得
3. **実績** — バッジ一覧（解除済み / 未解除）
4. **履歴** — 過去の運動記録一覧

---

## ゲームロジック

### 経験値・レベル
- 運動1セット完了 → 運動ごとに固定XPを付与
- レベルアップ必要XP：`現在レベル × 100 XP`
  - Lv1 → 2：100XP
  - Lv2 → 3：200XP
  - 以降同様
- レベル上限：なし

### キャラクター育成
レベルに応じてキャラの見た目（絵文字）が変化する。

| レベル | キャラ | 見た目 |
|--------|--------|--------|
| 1〜5   | ひよっこ | 🐣 |
| 6〜15  | 冒険者   | 🧍 |
| 16〜29 | 戦士     | ⚔️ |
| 30〜   | 勇者     | 🦸 |

### 実績バッジ

| バッジ名 | 解除条件 |
|----------|----------|
| 初めの一歩 | 初回記録 |
| 三日坊主克服 | 3日連続記録 |
| 週の勇者 | 7日連続記録 |
| 百戦錬磨 | 累計100回記録 |
| 筋肉王 | 腕立て伏せを累計50回記録 |

## データ構造（localStorage）

### `player`
```json
{
  "level": 5,
  "xp": 320,
  "totalXp": 820
}
```

### `records`（配列）
```json
[
  { "id": 1, "exerciseId": "pushup", "count": 10, "xp": 30, "date": "2026-03-29" }
]
```

### `badges`
```json
{
  "first_step": true,
  "3day_streak": false,
  "7day_streak": false,
  "100_times": false,
  "pushup_50": false
}
```

### `exercises`（コードに埋め込み、localStorageには保存しない）

`xpPerSet`：1セット完了時に得られるXP。単位（`unit`）は種目ごとに異なる。

```json
[
  { "id": "pushup",       "name": "腕立て伏せ",         "unit": "10回",  "xpPerSet": 30 },
  { "id": "squat",        "name": "スクワット",           "unit": "10回",  "xpPerSet": 25 },
  { "id": "plank",        "name": "プランク",             "unit": "30秒",  "xpPerSet": 20 },
  { "id": "jumping_jack", "name": "ジャンピングジャック", "unit": "10回",  "xpPerSet": 20 },
  { "id": "situp",        "name": "腹筋",                 "unit": "10回",  "xpPerSet": 25 },
  { "id": "momomage",     "name": "腿上げ",               "unit": "10回",  "xpPerSet": 20 }
]
```

## UX・エラー処理

### UX
- 運動記録後に「レベルアップ！」「バッジ獲得！」のポップアップ表示
- 経験値バーはCSSアニメーションで増加
- レスポンシブ対応（スマホでも使いやすいレイアウト）
- データリセットボタンを画面隅に配置

### エラー処理
- localStorage使用不可環境：起動時に検出しアラート表示
- データ破損（JSON parseエラー）：初期データにリセットして続行
